# MVP vs Advanced Features Roadmap

### Week 1-2: Foundation
- [x] User authentication (register, login, logout)
- [x] Basic household profile setup
- [x] Database schema implementation
- [x] API structure setup

### Week 3-4: Bill Management
- [x] Add/edit/delete bills
- [x] Bill categories (all 7 types)
- [x] Mark bills as paid/pending
- [x] Basic bill listing with filters

### Week 5-6: Dashboard
- [x] Total monthly expenses display
- [x] Upcoming bills widget
- [x] Category breakdown (pie chart)
- [x] Recent bills list
- [x] Basic monthly trend (line chart)

### Week 7-8: Polish & Deploy
- [x] Responsive design
- [x] Form validations
- [x] Error handling
- [x] Basic alerts for due dates
- [x] Deployment setup
